このAIによるレポートを人格データとしてSNSのユーザーの性質情報に書き込みタイムラインを各種形成します。

反応するかどうかまたどんな反応をするかをSNSの指標を使い定型化します。

### 反応対応表
|反応 |必要条件 |
|-----|----------|
|無視 |0 |
|いいね |好悪度+20 |
|リツイート |有効性+30  好悪度+30 |
|コメント |好悪度+50 or 好悪度-10 |


### パラメーター表
|名前 |説明 |
|---------|--------------|
|好悪度 |好感度と険悪度を示す指標 -であるほど険悪 +であるほど好き |
|有効性 |この投稿が自身のいるコミュニティーで役立つか、他の人にも伝えたいかなどの伝えたい欲求のことを指す |
